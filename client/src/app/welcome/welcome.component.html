Instructiuni

Folder upload: gastro.dev:3000/api/upload

Vagrant
  • vagrant ssh - conectare la vm - din folderul vm
  • vagrant ssh -c "cd /app && npm start" - conectare la masina virtuala si deschidere aplicatie in browser
  • vagrant up - porneste masina virtuala
  • vagrant halt - opreste masina virtuala
  • vagrant reload - restarteaza vm
  • vagrant destroy - distruge vm
  • vagrant fsnotify - urmareste modificarile din fisiere


Sequelize
http://docs.sequelizejs.com/manual/installation/getting-started.html
  • sequelize migration:create __name="nume fisier" - create migrations
  • sequelize db:migrate - run migrations
